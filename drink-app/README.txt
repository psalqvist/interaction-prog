# Clone project from git repo
1. Go to https://gits-15.sys.kth.se/fpe4/Group-11
2. Copy the SSH-key
3. Open the terminal
4. Use cd command to go to the directory where you want to clone the project
5. Write "git clone 'SSH-KEY'" in the terminal to clone the project
6. cd into the new folder

# Start the app
7. If you haven't installed Node.js and npm, go to https://nodejs.org/en/download/ and do so
8. Once you are in the root directory of the cloned project, and have npm installed, write the following in the terminal: npm start
9. The app should now start to run in your browser
